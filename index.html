株式会社コメダ

Corporate Overview

本社住所愛知県名古屋市東区葵三丁目12番23号
設立1975年8月（創業：1968年）
従業員数422名（連結：2024年2月末現在）
上場区分東証プライム・名証プレミア

1968年創業。個人経営の喫茶店から始まり、1975年8月に株式会社コメダ珈琲店を設立。1993年4月にFC展開を本格化し株式会社コメダを設立されました。

2014年に持株会社である株式会社コメダホールディングスが設立され、2016年6月に東京証券取引所市場第一部に上場。12月に名古屋証券取引所第一部へ上場しています。（現在はプライム市場）

現在、『珈琲所コメダ珈琲店』899店舗と、和風喫茶の『甘味喫茶おかげ庵』11店舗を展開。更に新しい業務形態として、ベーカリー型店舗『石窯パン工房ADEMOK』を沖縄県・南風原に1店舗、新業態の喫茶店『KOMEDA is □』（コメダイズ）を東京・東銀座に1店舗展開しています。

1968年の創業以来、 ″珈琲を大切にする心から” を信条に喫茶文化の根強い名古屋を中心に“くつろぎの場”を提供しています。フルサービス型喫茶店の全国チェーン展開として、店舗数・売上高は日本最大級となっています。くつろげる店づくり・自社製造商品・独自のFC運営システム等の強みにより、市場で独自のポジションを築いています。

日本を代表する喫茶文化の担い手。圧倒的なブランド力と独自のFCモデルで高収益を実現しています。安定基盤の上で次なる成長を牽引する、非常に魅力的なフェーズでの参画となります。

代表プロフィール

代表取締役社長　甘利 祐一 氏
1985年三和銀行(現三菱UFJ銀行)入行。サミー、タイヨーエレック代表取締役社長、セガサミーHD取締役等を経て2019年コメダ顧問。2022年5月代表取締役社長就任。

代表取締役会長　臼井 興胤 氏
三和銀行、セガ、ナイキ、日本マクドナルドCOO、セガ社長、グルーポン東アジア統括副社長を歴任。2013年コメダ社長就任、2022年より会長職。

株式会社ベネッセスタイルケアグループ

Corporate Overview

本社住所東京都新宿区西新宿2-3-1 新宿モノリスビル5F
設立2025年1月6日（中間持株会社として）
従業員数グループ全体 約20,000名規模
上場区分非上場（ベネッセHD傘下 / 将来的な独立運営視野）

2025年1月6日、ベネッセホールディングスの介護・保育領域を統括する中間持株会社として誕生。「介護と保育の未来を変える、新たなステージへ」を掲げ、地域密着型のサービスと事業成長を両立する次世代型ヘルスケアグループです。

ベネッセグループ内で教育・介護の両輪を明確に分け、介護領域を独立した経営体として再編。特に介護周辺領域での新規事業やM&Aを加速し、将来的な独立運営・エグジットも視野に入れた体制構築が進められています。

特にベネッセキャリオス（人材事業）、ベネッセパレット（介護食事業）を今後の成長ドライバーと位置づけ、組織体制・人材採用を強化中です。

代表「小林仁」様は、森本がリクルート時代から懇意にさせていただいております。小林さま自らのご依頼での案件であり、小林様に直接提案をさせていただきます。単独でのEXITの可能性もございます！

主要グループ会社

  • ■ ベネッセスタイルケア：全国展開の介護施設運営（主力事業）
  • ■ ベネッセパレット：介護食事業。設立6年で売上18億、急成長中。
  • ■ ベネッセキャリオス：介護・医療人材HRプラットフォーム。

株式会社ONODERA USER RUN

Corporate Overview

本社住所東京都千代田区大手町1-1-3 大手センタービル5階
設立2016年11月
従業員数グループ全体 29,638名（2025年4月時点）
上場区分非上場（上場準備中 / N-1フェーズ）

2016年11月設立。海外人材を日本国内に紹介する事業を展開しています。最大の強みは、アジア８カ国に独自の日本語学校を設置し、学生の選抜から教育までを一貫して自社で行っている点です。これにより、単なる紹介にとどまらず、企業に「顔が見える人材」を提供できるという明確な優位性を築いています。

主に扱うのは「特定技能」の領域で、これまでに約5,500人の特定技能外国人材を日本に紹介しています。2025年売上60億を見込み、約6,000名の入国受け入れを行っています。現在は上場を見据え、さらなる売上拡大を実現するために、国内での営業開拓の強化が喫緊の課題となっています。

LEOC社の給食事業が祖業。非上場でありながら、100%株式を保有。横浜FC経営権を持ち、Food creationsをアサヒから買収するなど、極めて勢いのあるグローバルグループです。

エン株式会社

Corporate Overview

本社住所東京都新宿区西新宿6-5-1 新宿アイランドタワー
設立2000年1月
従業員数1,993名（連結／2024年3月末現在）
上場区分東証プライム

「人と企業のより良い出会いを支援する」というミッションのもと、インターネットを活用した求人・転職支援サービスを展開するリーディングカンパニーです。企業の採用成功だけでなく、「入社後の活躍」にも焦点を当てた独自の支援が特徴です。

創業者であり代表取締役社長である越智通勝氏が提唱する「CSA（Career Selectability＝キャリア自己選択能力）経営」という哲学を核心に据えており、社員を企業の成長を牽引する人材へと育て上げる独自の仕組みを持っています。

大変革期にあたり、次代の経営を担う人材を全方位で積極採用中です。越智氏の40年にわたる経営ノウハウを直接受け継ぎ、共に汗を流せる稀有なポジションです。

代表プロフィール：越智 通勝 氏

1951年生まれ。1983年に日本ブレーンセンター設立。2000年にエン・ジャパンとして独立させ、わずか1年6カ月で上場。独自の経営思想「CSA経営」を実践し続けています。

株式会社シーユーシー

Corporate Overview

本社住所東京都港区芝浦3-1−1 msb Tamachi 田町ステーションタワーN 15階
設立2014年8月
従業員数連結 3,743名（2024年3月末現在）
上場区分東証グロース（エムスリーグループ）

「医療という希望をつくる」をミッションに掲げ、国内外の医療機関に対し経営コンサルティング／運営支援を行っています。東証プライム上場エムスリー社のグループ企業であり、2023年6月に上場。

主軸の病院経営支援に加え、訪問看護、ホスピス、さらには米国でのクリニック経営（34拠点）と、国内外で圧倒的なスピードで成長。現在はAI・DXを活用した「ヘルスケア総合カンパニー化」を目指し、IT×ヘルスケア領域の新会社設立を進めています。

経営陣との強固なリレーションがあり、過去複数名の入社実績がございます。今回はIT×ヘルスケア領域の新会社社長/COO候補という、非常に高い裁量と期待が込められた特注案件です。

代表プロフィール：濵口 慶太 氏

名古屋大学卒、ミシガン大学MBA取得。リクルート、ジェイ・ウィル・パートナーズ、エムスリーを経て、2014年CUC設立。変革を継続する姿勢「Change Until Change」を体現しています。
